Google Home Integration

Control BudgetFlow with your voice via Google Home and IFTTT.

How it works

  1. 1
    Generate your personal Webhook API Key below. This key is unique to your account and authenticates requests from IFTTT.
  2. 2
    Create an IFTTT Applet. Use Google Assistant as the trigger ("If you say…") and Webhooks as the action.
  3. 3
    Configure the Webhooks action with:
    • URL: {{ webhookUrl }}
    • Method: POST
    • Content Type: application/json
    • Body: {"action": "add_todo_item", "listName": "Chores", "text": "Clean bathroom", "repeatFrequency": "weekly"}
  4. 4
    Add your API key to the Webhooks action as the Authorization header value: Bearer YOUR_API_KEY.

Supported Actions

Action Required Fields Optional Fields
add_todo_item text, listId or listName completeByDate, repeatFrequency
add_item text, listId or listName —
create_list text (list name) listType, completeByDate, repeatFrequency
mark_todo_done occurrenceId, OR text + date listId, listName
mark_item_done listId or listName, plus text or itemId —
add_expense amount category
check_balance — —

Field Reference

Field Type Notes
action string One of the actions above
listId string MongoDB ObjectId of the list (fastest)
listName string Case-insensitive search (use when listId unknown)
text string Item text, list name for create_list, or item search text
listType string "todo" | "shopping" | "general" (default: "todo")
completeByDate string YYYY-MM-DD or RFC3339
repeatFrequency string daily | weekly | biweekly | monthly | yearly
occurrenceId string MongoDB ObjectId of a todo_occurrence document
itemId string List item's id field (hex string, not the list's _id)
date string YYYY-MM-DD for occurrence lookup
amount number Dollar amount for add_expense
category string Expense category
